# urban-heat-island-ml
Predicting Urban Heat Island intensity using satellite data, GIS analysis, and machine learning.
Urban Heat Island Prediction using Satellite Data & Machine Learning
This project analyzes urban heat island patterns in Delhi using satellite imagery and machine learning.
Workflow
Satellite preprocessing using Google Earth Engine
NDVI (vegetation) extraction
NDBI (built-up index) extraction
Land Surface Temperature calculation
Raster processing in QGIS
Pixel dataset generation
Random Forest machine learning model
Results
Model Performance:
R² = 0.874
RMSE = 0.0149
Tools Used
Google Earth Engine
QGIS
Python
Scikit-learn
Pandas
Matplotlib
